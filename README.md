# FastGANwithGrassmannSeed
基于 FastGAN 的改进：引入 Grassmann 流形初始化模块，提升生成稳定性与结构一致性。
✨ 项目简介

本项目基于 baseline FastGAN 进行改进，在生成器的起始阶段引入更结构化、更稳定的初始化，避免纯随机噪声在早期层面带来的不确定性与训练波动。为此，我们提出并实现了一个 流形初始化模块（Manifold / Grassmann Initialization），将输入噪声映射到 Grassmann 流形上的正交子空间表示，并投影生成初始特征图作为生成过程的“几何先验”。
本项目引入了 Grassmann 流形初始化机制。通过将随机噪声映射到正交子空间，我们为生成器提供了更强的“几何先验”，有效提升了生成图像的结构一致性与训练稳定性。
